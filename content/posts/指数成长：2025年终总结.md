# 前言

25 年是充实的一年。上半年大学毕业，下半年正式工作。回头看，这一年的节奏比想象中快，还好成长也比想象中快。

# 工作

一整年都在美团 TSP 营销方向。上半年实习主要做大促保障，当时觉得价值有限，没想到下半年遇上外卖大战，实习期的产出全派上了用场。

下半年正式入职后压力不小。营销业务的特性决定了隔几天就要压测值班、大促值班，加上外卖大战背景，业务需求推进很快。好在后续人力集中到一个系统性专项，能沉下心做系统层面改造。这个项目解决了系统一个多年顽疾，避免了潜在的资损和客诉。技术能力和方案能力都有大幅提升，成功拿到了 BU「攻坚克难奖」。

# 成长

上半年忙毕业，下半年忙工作，没了大一大二那种纯粹学习的时间。完整梳理时间线： 上半年重新系统学习 Java，虽然在 AI 时代这显得有点多余，但古法编程进入心流的感觉是 AI Coding 无法替代的。

面对看不完的 AI 知识，我选择了 Only Claude Code 战略 —— 只学 Claude Code 相关的内容和技术栈。现在 CC 已经是生活中不可切割的一部分，这篇文章也有他的润色。

25 年读完 28 本书，圆满达成「25 年 25 本书」的 Flag。最喜欢的是《精力管理》，它改变了我很多生活习惯，甚至影响了为人处世的方式。书中我最想分享的观点： 为自己定义价值观，并 100% 践行。即使只选择一两个 （比如耐心、自律），也会对生活产生显著的正面影响。
