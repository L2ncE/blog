# LanLance's Blog

## 目录结构

```
blog/
├── config.toml          # 站点配置
├── content/             # 文章内容
│   ├── posts/           # 博客文章
│   ├── about/           # 关于页面
│   └── friends/         # 友链页面
├── static/              # 静态资源
├── themes/              # 主题
│   └── no-style-please/
└── templates/           # 自定义模板（可选）
```

## 使用方法

### 本地预览

```bash
zola serve
```
